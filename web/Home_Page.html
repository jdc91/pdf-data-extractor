Data Dictionary
Data Dictionary is the place where the data elements to be mapped and extracted are defined. Select this menu option to manage Data Dictionary elements / attributes.

Templates
Select the Templates menu option to Add, Edit and organize templates. b. A Template need to be defined first, before a specific document can be extracted.

Extract
Extract option leads to the screen where documents can be selected and uploaded for extraction. This is the option to be selected to extract a new document.

Users
User menu option is for Administrators to manage users. It gives options to add, edit, lock/unlock and assign privileges to users. Access to this menu option is restricted to Administrators.
